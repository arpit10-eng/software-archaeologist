# Software Archaeologist

Software Archaeologist is a FastAPI-based backend application that analyzes GitHub repositories and generates insights about their structure, architecture, dependencies, and overall code quality.

## Features

- Repository validation
- Repository cloning
- Primary language detection
- Framework detection
- Dependency detection
- Architecture detection
- API endpoint detection
- Code structure detection
- Dependency graph generation
- Repository complexity analysis
- Repository health score calculation

## Tech Stack

- Python
- FastAPI
- Uvicorn
- Requests
